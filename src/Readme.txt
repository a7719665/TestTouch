Read the following before using the files within this archive.This archive contains files that belong to the article, Multi-touch and gestures in Flash, posted on the Adobe Developer Connection:
http://www.adobe.com/devnet/flash/articles/multitouch_gestures.html

These samples demonstrate the use of multi-touch and gesture APIs.

MultitouchExample.as and GestureExample.as can be run as Adobe Flex or Adobe AIR applications by doing the following:

1. Copy the files into a Flex Builder or Flash Builder project.
2. Right-click and choose "Set as Default Application".
3. Run or debug the project.
